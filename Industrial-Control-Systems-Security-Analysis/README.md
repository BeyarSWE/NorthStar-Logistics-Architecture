# Operational Technology (OT) Security Project

This project is part of the Operational Technology (OT) course within the IT Security Engineering program. It focuses on analyzing risks, vulnerabilities, and security measures for Industrial Control Systems (ICS) in critical infrastructure.

## Objectives
* Analyze the technical differences and synergies between IT and OT systems.
* Evaluate operational security based on the NIS2 Directive and ISA/IEC 62443 standards.
* Develop incident response and business continuity strategies for industrial environments.
* Conduct a security assessment of a specific industrial case study.

## Methodology
The project follows professional standards for security documentation:
* **Version Control:** Git is used for tracking changes and peer review via Pull Requests.
* **Task Management:** GitHub Issues and Projects are utilized for workflow transparency.
* **Standards Compliance:** All recommendations are mapped against ISA/IEC 62443 and NIS2 requirements.

## Repository Structure
* **`docs/`**: Final report and individual examination responses.
* **`research/`**: Reference materials and analysis of regulatory frameworks.
* **`assets/`**: Architectural diagrams of SCADA, PLC, and DCS environments.
